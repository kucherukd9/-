Консольний чат-бот "Організатор подій"

Мета проєкту

Метою даного проєкту є створення консольного чат-бота на мові програмування Python, який допомагає користувачу організовувати події. Бот дозволяє додавати нові події, переглядати список подій, редагувати їх, видаляти та переглядати події за певними датами.

Опис логіки та алгоритму роботи бота

При запуску програми бот зчитує дані з файлу events.json. У цьому файлі зберігається список усіх подій. Після цього користувачу виводиться список доступних команд.

Користувач вводить команду в консоль, а програма обробляє її за допомогою відповідної функції. Наприклад, команда "додати" викликає функцію додавання нової події. Бот запитує назву події, дату, час та категорію.

Після додавання, редагування або видалення події список автоматично зберігається у файл. Програма працює у циклі до тих пір, поки користувач не введе команду "вийти".

Опис структури даних

Події зберігаються у файлі формату JSON. Дані представляються у вигляді списку словників. Кожна подія містить назву, дату, час та категорію.

Приклад структури події:

{
"name": "Екзамен",
"date": "2026-06-10",
"time": "10:00",
"category": "університет"
}

Інструкція запуску програми

1. Встановити Python на комп’ютер.
2. Завантажити файли проєкту.
3. Відкрити папку з проєктом у терміналі або середовищі розробки.
4. Запустити програму командою:

python bot.py

Введіть команду: додати

Дата (YYYY-MM-DD): 2026-05-15  
Час (HH:MM): 10:00  
Категорія: університет
